# 🛍️ Customer Shopping Behaviour Analysis using Python, PostgreSQL & Power BI

> **Transforming retail customer data into actionable business insights through data analytics, SQL, and interactive dashboarding.**

## 📌 Project Overview

This project presents an end-to-end **Customer Shopping Behaviour Analysis** solution built using **Python**, **PostgreSQL**, and **Power BI**. The objective is to analyze customer purchasing patterns, identify key business trends, and provide actionable insights that support data-driven decision-making.

The project follows a complete analytics workflow—from **data cleaning and preprocessing** to **exploratory data analysis (EDA)**, **SQL-based business intelligence**, and **interactive Power BI dashboard development**. By transforming raw retail data into meaningful visualizations and business metrics, the project demonstrates how data can be leveraged to improve customer understanding and business performance.

---

## 🎯 Business Objectives

- Clean and preprocess raw customer shopping data.
- Handle missing values and validate data quality.
- Perform Exploratory Data Analysis (EDA).
- Solve real-world business questions using SQL.
- Develop an interactive Power BI dashboard.
- Generate actionable business insights and recommendations.

## 💡 Key Insights

- Clothing contributes the highest overall sales.
- Adult customers represent the largest customer segment.
- Loyal customers account for a significant share of repeat purchases.
- Express shipping customers tend to spend slightly more.
- High-rated products present strong opportunities for targeted promotions.
